## The Problem

Every major file converter (CloudConvert, Zamzar, ILovePDF, Smallpdf) silently strips VBA macros from XLSM and DOCM files. They do this deliberately — running macros at scale is a security liability.

For enterprise users migrating legacy Excel workbooks, this is a hard blocker with no obvious solution.
