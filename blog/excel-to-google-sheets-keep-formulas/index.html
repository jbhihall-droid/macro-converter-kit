What Happens When You Import Excel to Google Sheets

Google Sheets accepts XLSX, XLS, and XLSM files via upload. The import engine converts most content, but there are predictable failure points. Knowing them before you migrate saves hours of debugging.
XLSM note: Macros (VBA) are stripped on import — Google Sheets uses Apps Script, not VBA. If macros are critical, see our XLSM to Google Sheets macro guide.

Step-by-Step Import

  1. Open Google Drive → click + New → File upload
  2. Upload your XLSX or XLSM file
  3. Double-click the uploaded file in Drive — it opens as a Sheets document
  4. OR: right-click → Open with → Google Sheets → it auto-converts
  5. Check the formula error list that appears (if any)

Common Formula Errors After Import

#NAME? errors — Excel functions not available in Sheets (STOCKHISTORY, WEBSERVICE, some statistical functions). Replace with Sheets equivalents. XLOOKUP → not available — Replace with INDEX/MATCH in Sheets or use VLOOKUP. Array formula syntax — Excel Ctrl+Shift+Enter arrays become single-cell in Sheets. Use ARRAYFORMULA() wrapper. 
# Excel:
{=SUM(IF(A1:A10>5, A1:A10, 0))}

# Google Sheets equivalent:
=SUMIF(A1:A10, ">5", A1:A10)

Automating XLSX → Sheets via Python + gspread

import gspread
from google.oauth2.service_account import Credentials
import pandas as pd

# Read Excel
df = pd.read_excel('report.xlsx')

# Connect to Google Sheets
creds = Credentials.from_service_account_file('service_account.json',
    scopes=['https://www.googleapis.com/auth/spreadsheets'])
gs = gspread.authorize(creds)

# Create or open sheet
spreadsheet = gs.create('Imported Report')
worksheet = spreadsheet.sheet1
worksheet.update([df.columns.tolist()] + df.values.tolist())
print('Uploaded to Google Sheets')

Frequently Asked Questions

Does Excel to Google Sheets import preserve all formulas?

Most standard formulas (SUM, IF, VLOOKUP, INDEX/MATCH) import correctly. Excel-exclusive functions like XLOOKUP, STOCKHISTORY, and Power Query connections do not have direct equivalents in Sheets and will show errors.

Can I import XLSM macros to Google Sheets?

No. VBA macros are stripped on import. Google Sheets uses Apps Script (JavaScript-based) as its macro language. You need to rewrite any critical macros in Apps Script after import.

What is the file size limit for importing Excel to Google Sheets?

Google Sheets supports imports up to 100MB for uploaded files. Very large Excel files (100K+ rows) may import slowly or hit cell limits (10 million cells per spreadsheet).
