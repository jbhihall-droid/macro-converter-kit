XLSM to XLSX: What You Lose and How to Convert Safely

By MacroKit  ·  Updated April 2026  ·  5 min read

Why Convert XLSM to XLSX?

XLSM files are Excel's macro-enabled format. XLSX is the standard, widely-compatible format without macro support. You'd convert XLSM → XLSX when sharing with people who don't need the macros, uploading to systems that reject macro-enabled files, or reducing file size and compatibility concerns.
Warning: Excel will permanently delete all VBA code when saving XLSM as XLSX. There is no undo once the file is saved in a new location. Always keep the original XLSM.

What Gets Lost in the Conversion

ElementSurvives XLSM → XLSX?
Cell data, formulas, chartsYES
Pivot tables, named rangesYES
Conditional formattingYES
VBA macros (all modules)NO — permanently deleted
Form controls linked to macrosNO — buttons become static
Custom ribbon tabs (VBA-driven)NO
ActiveX controlsNO

Step-by-Step: XLSM to XLSX in Excel

  1. Back up your XLSM first — copy it somewhere safe before proceeding
  2. Open the XLSM file in Excel
  3. Click File → Save As
  4. From the dropdown, select Excel Workbook (*.xlsx)
  5. Excel warns: "The following features cannot be saved in macro-free workbooks" — click Continue
  6. Save. The XLSX has no macros. Your original XLSM is untouched if you kept it.
Python alternative (openpyxl):
import openpyxl

# Open the XLSM (macros are ignored)
wb = openpyxl.load_workbook('report.xlsm', keep_vba=False)
# Save as XLSX
wb.save('report.xlsx')
print('Converted — all cell data preserved, VBA removed')

Batch Convert Multiple XLSM Files to XLSX

For bulk conversion without opening each file: 
import glob, openpyxl, os

for path in glob.glob('/path/to/files/*.xlsm'):
    wb = openpyxl.load_workbook(path, keep_vba=False)
    out = path.replace('.xlsm', '.xlsx')
    wb.save(out)
    print(f'Converted: {os.path.basename(path)} → {os.path.basename(out)}')

This processes an entire folder. Each XLSX will be identical in data to the source XLSM — minus VBA.

When You Need to Keep the Macros

If the macros are essential, don't convert — distribute the XLSM with macro warnings documented. If the destination system rejects XLSM, export data-only sheets to XLSX and keep the XLSM as the operational version.
MacroKit's recommendation: Keep the XLSM as your source of truth. Create a macro-free XLSX export for sharing. Never delete the XLSM.

Frequently Asked Questions

Can I convert XLSM to XLSX without losing data?

Yes. All cell data, formulas, charts, and pivot tables survive the conversion. Only VBA macros and macro-linked form controls are removed. Use Excel's Save As or Python openpyxl with keep_vba=False.

Does renaming .xlsm to .xlsx work?

No. Renaming the extension does not change the file format. Excel will either reject the file or open it incorrectly. You must use File → Save As to properly rewrite the XML structure.

Can I recover macros after converting to XLSX?

No. Once saved as XLSX, the vbaProject.bin is permanently discarded. Always keep the original XLSM as backup before converting.
