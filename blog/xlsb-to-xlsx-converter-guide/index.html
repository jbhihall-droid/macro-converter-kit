XLSB to XLSX: Convert Binary Excel to Standard Format

Sponsored

What Is XLSB and Why Convert to XLSX?

XLSB (Excel Binary Workbook) is a compressed binary format that opens faster and saves with smaller file sizes than XLSX — but it's less portable. Many systems, APIs, and BI tools that accept XLSX reject XLSB. Converting to XLSX makes the file universally readable by Google Sheets, LibreOffice, Pandas, and every data tool that understands Open XML.
Key fact: XLSB can contain VBA macros (unlike XLSX). When you convert to XLSX, macros are permanently removed. If macros matter, convert to XLSM instead.

Method 1: Excel Save As (Simplest)

  1. Open the XLSB file in Excel
  2. File → Save As
  3. Format dropdown → Excel Workbook (*.xlsx)
  4. If macros exist, Excel warns they'll be removed — click Continue
  5. Save to new location. Original XLSB untouched.

Method 2: Python — openpyxl or xlrd

openpyxl does not support XLSB. Use pyxlsb to read, then openpyxl to write XLSX: 
from pyxlsb import open_workbook as open_xlsb
import openpyxl

# Read XLSB
rows = []
with open_xlsb('data.xlsb') as wb:
    with wb.get_sheet(1) as sheet:
        for row in sheet.rows():
            rows.append([item.v for item in row])

# Write XLSX
wb_out = openpyxl.Workbook()
ws = wb_out.active
for row in rows:
    ws.append(row)
wb_out.save('data.xlsx')
print('Converted to XLSX')

Install: pip install pyxlsb openpyxl

Batch Convert XLSB Folder to XLSX

import glob, os
from pyxlsb import open_workbook as open_xlsb
import openpyxl

for path in glob.glob('/path/to/*.xlsb'):
    rows = []
    with open_xlsb(path) as wb:
        with wb.get_sheet(1) as sheet:
            for row in sheet.rows():
                rows.append([item.v for item in row])
    out_path = path.replace('.xlsb', '.xlsx')
    wb_out = openpyxl.Workbook()
    ws = wb_out.active
    for row in rows:
        ws.append(row)
    wb_out.save(out_path)
    print(f'Done: {os.path.basename(out_path)}')

Frequently Asked Questions

Can openpyxl read XLSB files?

No. openpyxl does not support XLSB format. Use the pyxlsb library to read XLSB, then openpyxl to write the output as XLSX.

Does converting XLSB to XLSX lose any data?

Cell data, formulas, and most formatting survive. VBA macros do not — XLSX cannot contain macros. Pivot table definitions may also need reconnecting after conversion.

Is XLSB or XLSX better for large files?

XLSB opens faster and saves smaller for very large files (100K+ rows). XLSX is better for compatibility with non-Excel tools. For pure data interchange, CSV is often the best choice.
