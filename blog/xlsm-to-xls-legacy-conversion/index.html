XLSM to XLS: Convert Macro-Enabled Excel to Legacy Format

By MacroKit  ·  Updated April 2026  ·  5 min read

Why Convert XLSM to XLS?

XLS is the Excel 97-2003 binary format. Despite being 30 years old, you still need it for legacy ERP systems, old banking software, government portals, and industry-specific tools that haven't been updated. XLSM → XLS is a backward-compatibility migration.
Good news: XLS supports VBA macros via the .xls container. Unlike XLSX, macros can survive this conversion — but with caveats.

What Survives XLSM → XLS?

ElementSurvives?Notes
Cell data, formulasYESMostly — some new Excel functions not in XLS
VBA macrosPARTIALBasic VBA survives; newer API calls may break
ChartsMOSTLYOlder chart types convert; newer types may downgrade
Row limitCAPPEDXLS max is 65,536 rows vs 1,048,576 in XLSM
Column limitCAPPEDXLS max is 256 columns vs 16,384 in XLSM
Conditional formattingPARTIALSimple rules survive; newer rule types may not

Method 1: Excel Save As XLS

  1. Open your XLSM file in a modern Excel version
  2. File → Save As
  3. Format dropdown → Excel 97-2003 Workbook (*.xls)
  4. Excel compatibility checker runs — review any flagged issues
  5. Click Continue to save. Resolve flagged issues manually if they matter.

Method 2: xlwt (Python — XLS write library)

# For simple data export to XLS:
import xlwt, openpyxl

# Read XLSM
wb_in = openpyxl.load_workbook('report.xlsm', keep_vba=False)
ws_in = wb_in.active

# Write XLS
wb_out = xlwt.Workbook()
ws_out = wb_out.add_sheet('Sheet1')
for i, row in enumerate(ws_in.iter_rows(values_only=True)):
    for j, val in enumerate(row):
        ws_out.write(i, j, val if val is not None else '')
wb_out.save('report.xls')
print('Saved as XLS')

Row Limit Warning for Large Files

If your XLSM has more than 65,536 rows, the conversion to XLS will fail or truncate data silently. Always verify: 
import openpyxl
wb = openpyxl.load_workbook('report.xlsm', read_only=True)
ws = wb.active
if ws.max_row > 65536:
    print(f'WARNING: {ws.max_row} rows — XLS will truncate at 65,536')
else:
    print(f'{ws.max_row} rows — safe to convert to XLS')

Frequently Asked Questions

Do VBA macros survive when converting XLSM to XLS?

Basic VBA macros generally survive. However, macros using Excel object model features introduced after Office 2003 may break. Test all macro functionality after conversion in the target environment.

What is the maximum row count in XLS format?

XLS (Excel 97-2003) supports a maximum of 65,536 rows and 256 columns. If your XLSM file has more rows, data will be silently truncated. Modern XLSX/XLSM supports 1,048,576 rows.

Why would I use XLS in 2026?

Legacy systems that were built before 2007 (when XLSX was introduced) often require XLS input. Common examples: SAP interfaces, older government portals, financial reporting systems with fixed import formats.
