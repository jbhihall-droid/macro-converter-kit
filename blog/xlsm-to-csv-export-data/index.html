XLSM to CSV: Export Your Data Without Breaking Anything

Sponsored
By MacroKit  ·  Updated April 2026  ·  5 min read

Why Export XLSM to CSV?

CSV is the universal data format — databases, APIs, BI tools, and scripts all accept it. When your data lives in an XLSM (macro-enabled workbook), exporting to CSV strips the VBA but preserves all cell values. Common reasons: importing into databases, feeding Python/R data pipelines, sharing with non-Excel users, archiving data without macro dependencies.

What Happens to Your Data

ElementIn CSV Export
Cell values (text, numbers)Preserved
FormulasValues only (formula evaluated)
VBA macrosNot exported (expected)
Formatting, chartsNot in CSV format
Multiple sheetsOne sheet per CSV file

Method 1: Excel Save As CSV (Single Sheet)

  1. Open your XLSM in Excel
  2. Navigate to the sheet you want to export
  3. File → Save As → select CSV UTF-8 (Comma delimited) (*.csv)
  4. Excel warns about features not supported in CSV — click OK
  5. The CSV contains only the active sheet
Common issue: If you have dates or numbers formatted in locale-specific ways, they may export differently. Preview the CSV in a text editor before importing elsewhere.

Method 2: Python pandas — All Sheets or Batch

import pandas as pd

# Export a specific sheet
df = pd.read_excel('report.xlsm', sheet_name='Data', engine='openpyxl')
df.to_csv('report_data.csv', index=False)

# Export ALL sheets to separate CSVs
xl = pd.ExcelFile('report.xlsm', engine='openpyxl')
for sheet in xl.sheet_names:
    df = xl.parse(sheet)
    df.to_csv(f'report_{sheet}.csv', index=False)
    print(f'Exported: {sheet}.csv')

pandas with openpyxl reads XLSM files correctly. Macros are ignored — only data is read.

Batch Export: Folder of XLSM Files to CSV

import glob, pandas as pd, os

for path in glob.glob('/path/to/files/*.xlsm'):
    basename = os.path.splitext(os.path.basename(path))[0]
    xl = pd.ExcelFile(path, engine='openpyxl')
    for sheet in xl.sheet_names:
        df = xl.parse(sheet)
        out = f'{basename}_{sheet}.csv'
        df.to_csv(out, index=False)
        print(f'{out} — {len(df)} rows')

Frequently Asked Questions

Does exporting XLSM to CSV break the macros?

The macros are not exported to CSV — they're left in the XLSM. Your original file is unchanged. CSV is data-only format, so macros simply don't exist in that format.

Can I export all sheets from an XLSM to CSV at once?

Not via Excel's Save As (it only exports the active sheet). Use Python's pandas library to loop through all sheet names and export each to a separate CSV file.

What happens to formulas when exporting to CSV?

Formula results (calculated values) are preserved in the CSV. The formula text is not — CSV stores the evaluated number or text, not the Excel formula.
